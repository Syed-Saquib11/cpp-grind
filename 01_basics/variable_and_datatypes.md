Variable are representation of some numbers or value.
There are two steps of creating and using a variable:
    1. Declaration
    2. Assignment

    Declaration:
        We need to first declare the type of the variable (number, character, string, etc.)
        ```
        int x;
        ```
    Assignment:
        Here we assigns a value to our variable
        ```
        x = 5;
        ```
    We can also both declare and assign at the same time:
        ```
            int y = 6;
            int sum = x + y;
        ```

Types of Datatypes:
    Int -> Whole integers
    Double -> Number including decimal
    Char -> Single Character
    Booleans -> True or False
    Strings -> Objects that represents a sequence of text, Char but more than one character, even whole sentences. ex: address. they are provided by std namespace
